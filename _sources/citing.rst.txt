Citing Reticula
===============

If this software was directly or indirectly useful in your reasearch, you can
cite the pre-print:

.. code-block:: BibTex

   @article{badie2022reticula,
     doi = {10.48550/ARXIV.2207.10771},
     url = {https://arxiv.org/abs/2207.10771},
     author = {Badie-Modiri, Arash and Kivelä, Mikko},
     title = {Reticula: A temporal network and hypergraph analysis software package},
     publisher = {arXiv},
     year = {2022}
   }

.. code-block:: text

   Badie-Modiri, Arash and Mikko Kivelä. "Reticula: A temporal network and hypergraph analysis software package." arXiv preprint arXiv:2207.10771 (2022)
